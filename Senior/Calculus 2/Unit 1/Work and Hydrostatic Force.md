# Work
$W~\text{(Work)}=F~\text{(Force)}*d~\text{(Distance)}$

Work done by a variable force $F(x)$ to move an object along the $x$-axis from $x=a$ to $x=b$ is: $$W=\int^b_a F(x)~dx$$
